setFinishedMotionHandler

  • setFinishedMotionHandler(onFinishedMotionHandler?: ((self: ACubismMotion) => void)): void

  • モーション再生終了コールバックの登録

    モーション再生終了コールバックを登録する。 isFinishedフラグを設定するタイミングで呼び出される。 以下の状態の際には呼び出されない:
